Overview

Depth.zip is a lightweight monocular depth estimation network designed for zero-shot cross-domain generalisation on resource-constrained hardware — from embedded processors to mobile NPUs and smartphone neural engines.

The model is trained entirely via knowledge distillation from a large-scale teacher network, without ever seeing metric ground-truth depth at training time. At inference, it produces full-resolution affine-invariant depth maps in real time on iPhone, with no domain-specific fine-tuning required.

A hardware-adaptive upsampling module provides two computational paths — one optimised for GPU/TensorRT and one for mobile NPUs (Apple Neural Engine, Qualcomm AI Engine) — allowing the same model to be deployed across heterogeneous devices without retraining.

Live on Device

The model runs entirely on-device via Apple Neural Engine — no internet connection required.
